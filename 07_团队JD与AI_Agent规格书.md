# 07 团队JD与AI Agent规格书

> 本文档定义每个"作战单位"的职责边界、输入输出和考核标准
> 用途：招聘/培训人员 + 设计/部署AI Agent

---

## 第一部分：人类团队JD

### 岗位1：项目总指挥（你）

**岗位概述：** 整个项目的战略决策者和品牌方对接人。不直接生产内容，但决定内容方向。

| 维度 | 内容 |
|------|------|
| 汇报对象 | 品牌方 |
| 管理对象 | 5名内容运营 + 7个AI Agent |
| 每日工时 | 2-3小时 |
| 核心职责 | 1. 每日策略决策（确定次日内容方向）<br>2. 品牌方审核沟通与关系维护<br>3. 预算管控与投流总策略<br>4. 每周战略复盘主持<br>5. 异常情况应急处理 |
| KPI | GMV达成率、预算使用效率(ROI)、品牌方满意度 |
| 必备能力 | 数据分析意识、沟通协调、快速决策 |
| 不需要 | 剪辑技术、AI工具操作（由团队执行） |

---

### 岗位2：内容运营（5人，每人管3个账号）

**通用JD（5人共享）：**

| 维度 | 内容 |
|------|------|
| 汇报对象 | 项目总指挥 |
| 管理对象 | 3个账号（抖音+视频号+小红书各1） |
| 每日工时 | 4-5小时 |
| 核心职责 | 1. 用AI工具生产当日内容（脚本→图片→视频→合成）<br>2. 内部质检（合规+差异化+品牌一致性）<br>3. 按排期发布内容<br>4. 评论区互动回复<br>5. 每日数据回收填表<br>6. 每周拍30分钟实拍素材 |
| KPI | 日均内容产出量、内容合格率(>95%)、账号GMV贡献 |
| 必备能力 | AI工具基础操作、剪映基础、中文写作、细心 |
| 培训周期 | 1-2天（Day 0-1 跑通全流程即可上手） |

**5人的差异化职责：**

| 成员 | 额外角色 | 额外职责 | 额外KPI |
|------|---------|---------|---------|
| **灏灏** | 数据分析员 | 每晚汇总全组数据，输出"次日生产指令" | 数据报告及时率100% |
| **家豪** | 投流操盘手 | 统一管理15个账号的DOU+/薯条投放 | 整体投流ROI ≥ 1:2 |
| **小瓶子** | 实拍素材总监 | 每周组织拍摄+管理共享素材库 | 素材库每周更新 |
| **和煦好物** | 质检总监 | 每天抽查10条内容的合规性 | 违规发布事故为0 |
| **泽鑫** | 品牌审核对接 | 每日提交审核+跟进结果+处理打回 | 审核通过率 ≥ 80% |

---

### 岗位3：如需招聘的补充人手

如果5人团队在第二周发现工作量过大，优先补充以下岗位：

**补充岗位A：剪辑助理（兼职/实习）**
- 职责：专门负责剪映批量合成，释放5人去做策略和运营
- 工时：每天3小时
- 要求：会用剪映、细心
- 优先级：高（最能解放产能的岗位）

**补充岗位B：数据录入员（兼职）**
- 职责：每日数据回收填表、整理周报数据
- 工时：每天1小时
- 要求：会用飞书/Excel、细心
- 优先级：中

## 第二部分：AI Agent规格书

### Agent 1：策略分析师

| 维度 | 规格 |
|------|------|
| 底层模型 | Claude Opus/Sonnet |
| 触发频率 | 每日1次（22:00数据复盘后） |
| 输入 | 当日全量数据表（CSV/JSON） |
| 输出 | 次日生产指令（JSON格式） |
| 核心能力 | 1. ABCD四象限分类<br>2. 模板效果排序<br>3. 投流ROI分析<br>4. 异常检测（如某账号突然限流）<br>5. 生成次日内容方向建议 |
| 自进化机制 | 每周更新分析prompt，加入本周新发现的规律 |
| 人类监督 | 项目总指挥审核分析结果后确认 |

**系统提示词核心片段：**
```
你是世宝吉项目的数据策略分析师。
基于今日数据，完成以下分析：
1. 将所有内容按自然流表现×投流ROI分为ABCD四象限
2. 统计各模板类型的平均转化率，排序
3. 识别表现异常的账号（突然下降/上升）
4. 输出明日生产指令（模板配比、产品重心、投流建议）
输出格式为结构化JSON。
```

---

### Agent 2：文案工厂

| 维度 | 规格 |
|------|------|
| 底层模型 | Claude Opus/Sonnet |
| 触发频率 | 每日1次（9:00生产时段） |
| 输入 | 品牌信息+产品信息+当日策略指令+爆款基因库 |
| 输出 | 50条结构化脚本（JSON） |
| 核心能力 | 1. 按5个模板类型生成脚本<br>2. 自动嵌入安全话术<br>3. 自动规避禁用词<br>4. 基于爆款基因库重组新脚本<br>5. 适配三个平台调性 |
| 自进化机制 | 每周将新的爆款基因写入系统prompt |
| 质量门 | 每条输出自动标注合规检查结果 |

---

### Agent 3：视觉设计师

| 维度 | 规格 |
|------|------|
| 底层工具 | Nano Banana 2.0 |
| 触发频率 | 每日1次（9:30） |
| 输入 | 品牌素材图+场景提示词（来自Agent 2） |
| 输出 | 50-80张场景图/封面图 |
| 核心能力 | 1. 产品场景图生成<br>2. 小红书封面图生成<br>3. 品牌风格一致性<br>4. 多角度/多场景变体 |
| 自进化机制 | 根据封面点击率数据优化提示词 |
| 品牌约束 | 色调#C92029/#FFF500/#2E7D32/#F5F0E6 |

---

### Agent 4：视频导演

| 维度 | 规格 |
|------|------|
| 底层工具 | Seedance 2.0 |
| 触发频率 | 每日1次（10:00） |
| 输入 | Agent 3的场景图+动效提示词 |
| 输出 | 30-50段动态视频素材（5-10秒/段） |
| 核心能力 | 1. 产品展示动效<br>2. 倒饮/开箱动态<br>3. 原料到产品的转化动效<br>4. 品牌质感维持 |
| 自进化机制 | 根据完播率数据优化动效风格 |

---

### Agent 5：配音演员

| 维度 | 规格 |
|------|------|
| 底层工具 | 剪映AI配音 / 魔音工坊 |
| 触发频率 | 随脚本生产实时 |
| 输入 | 脚本文本+音色指令（5种固定人设音色） |
| 输出 | 配音音频文件 |
| 预设音色 | 灏灏:知性女声 / 家豪:年轻男声 / 小瓶子:温柔女声 / 和煦好物:磁性男声 / 泽鑫:成熟男声 |

---

### Agent 6：合规检察官

| 维度 | 规格 |
|------|------|
| 底层模型 | Claude（轻量调用） |
| 触发频率 | 每条文案生成时自动运行 |
| 输入 | 文案文本 |
| 输出 | PASS/FAIL + 违规词标记 |
| 检测词库 | 治疗/根治/预防/止咳/化痰/消炎/平喘/降压/降糖/润肺/理气/调理/修复/增强免疫/最佳/顶级/第一/特效/秒见效/必改善/无毒副作用/医生推荐/临床验证 |
| 准确率目标 | 99%+（宁可误杀不可漏放） |

---

### Agent 7：数据监控员

| 维度 | 规格 |
|------|------|
| 底层工具 | 蝉妈妈API + 飞书多维表格 |
| 触发频率 | 每3小时自动更新 |
| 输入 | 各平台账号数据 |
| 输出 | 实时数据仪表盘 |
| 监控指标 | 播放/完播率/互动/商品点击/下单/GMV |
| 预警规则 | 某账号连续3条播放<100 → 可能被限流预警 |
| 局限 | 部分平台数据可能需人工补充 |

## 第三部分：人类×AI协作总图

```
项目总指挥（你）
  │
  ├─ 每晚22:00 ── 审核Agent 1（策略分析师）的建议 → 确认次日方向
  ├─ 每日17:00 ── 审核泽鑫提交给品牌方的审核包
  ├─ 每周六 ──── 主持全员复盘会
  │
  ├── 灏灏（数据+科普线）
  │    ├── 操作 Agent 2 生成科普脚本
  │    ├── 操作 Agent 3+4 生成视觉素材
  │    ├── 每晚运行 Agent 1 生成分析报告
  │    └── 管理：抖音科普号 / 视频号家庭养生 / 小红书成分党
  │
  ├── 家豪（投流+职场线）
  │    ├── 操作 Agent 2 生成职场脚本
  │    ├── 统一操作全组15个号的投流（DOU+/薯条）
  │    └── 管理：抖音办公室养生 / 视频号职场健康 / 小红书体验记录
  │
  ├── 小瓶子（实拍+生活线）
  │    ├── 操作 Agent 2 生成生活方式脚本
  │    ├── 每周拍摄实拍素材30分钟 → 共享素材库
  │    └── 管理：抖音好物分享 / 视频号家庭养生 / 小红书精致生活
  │
  ├── 和煦好物（质检+测评线）
  │    ├── 操作 Agent 2 生成测评脚本
  │    ├── 每日抽查10条内容 → 运行 Agent 6 合规检查
  │    └── 管理：抖音测评家 / 视频号品牌文化 / 小红书精致生活
  │
  └── 泽鑫（审核对接+送礼线）
       ├── 操作 Agent 2 生成送礼脚本
       ├── 每日整理全组内容 → 提交品牌方审核 → 跟进结果
       └── 管理：抖音送礼情报站 / 视频号职场健康 / 小红书送礼攻略
```
