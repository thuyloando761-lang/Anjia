# 第一部分：三大平台运营规则

## 一、抖音运营规则

### 1.1 账号基础要求

| 项目 | 要求 |
|------|------|
| 开通商品橱窗 | 粉丝≥1000 + 实名认证 + 缴纳500元保证金 |
| 短视频挂车 | 开通橱窗后即可在视频中挂商品链接 |
| 直播带货 | 粉丝≥1000（你们暂不做直播，但留作备选） |
| 每日发布上限 | 官方无硬上限，但建议单号不超过10条/天 |

### 1.2 内容规则（红线）

**会被限流的行为：**
- 搬运/重复内容（同一视频发多个号、同模板无差异化复制）
- 过度营销（整条视频都是广告，无任何信息价值）
- 低质画面（模糊、黑屏、纯文字滚动）
- 违规引流（在视频或评论区留微信号、手机号）
- 涉及医疗功效宣称（对你们来说最大风险）
- 频繁删除重发视频

**会被封号的行为：**
- 多次违规被警告后继续违规
- 虚假宣传被用户投诉
- 刷量/买粉（绝对不要做）
- 账号之间互相导流被判矩阵号

**AI内容的特殊风险：**
- 抖音已经可以识别部分AI生成视频
- 目前政策：AI内容不被禁止，但必须标注"AI生成"（在描述或视频中）
- 建议：混入20-30%实拍素材降低纯AI判定风险
- 一旦发现限流，立刻减少纯AI内容比例

### 1.3 算法偏好（影响推流的关键指标）

| 指标 | 权重 | 健康值 | 说明 |
|------|------|--------|------|
| 完播率 | 极高 | ≥25% | 最重要的指标，决定是否进入下一级流量池 |
| 互动率 | 高 | ≥3% | 点赞+评论+转发+收藏 / 播放 |
| 转发率 | 高 | ≥0.5% | 转发是比点赞更强的推流信号 |
| 关注率 | 中 | ≥1% | 看了视频后关注你的比例 |
| 商品点击率 | 中 | ≥2% | 带货视频的额外考量指标 |

**推流机制：**
```
初始流量池（200-500播放）
  ↓ 完播率>25% + 互动率>3%
二级流量池（3000-5000播放）
  ↓ 持续表现好
三级流量池（1万-10万播放）
  ↓ 极优表现
爆款池（10万+播放）
```

### 1.4 DOU+投放规则

- 最低投放金额：100元
- 投放目标选项：系统智能推荐 / 点赞评论 / 粉丝增长 / 商品点击 / 商品购买
- **你们应该选"商品购买"或"商品点击"** — 不要选"点赞评论"（虚荣指标）
- 审核时间：通常1-2小时，偶尔会被拒（通常是内容违规）
- 投放人群：可以定向年龄/性别/地域/兴趣 → 建议定向30-55岁+养生兴趣标签
- 投放时段：建议选择12-14点、19-22点（用户活跃高峰）

### 1.5 抖音带货的关键时间节点

| 时段 | 流量特征 | 建议 |
|------|---------|------|
| 7:00-9:00 | 早高峰通勤 | 适合发15秒内的短平快内容 |
| 12:00-14:00 | 午休高峰 | 第一波发布黄金时段 |
| 17:00-19:00 | 晚高峰过渡 | 适合发科普类较长内容 |
| 19:00-22:00 | 晚间黄金 | 第二波发布最佳时段，也是投流效果最好的时段 |
| 22:00-24:00 | 深夜长尾 | 完播率通常最高（用户刷得慢） |

## 二、小红书运营规则

### 2.1 账号基础要求

| 项目 | 要求 |
|------|------|
| 开通商品笔记 | 粉丝≥1000 + 实名认证 + 开通专业号 |
| 笔记挂链接 | 开通商品笔记功能后可挂商品链接 |
| 每日发布上限 | 建议不超过5篇/天（小红书对高频发布更敏感） |
| 内容审核 | 比抖音更严格，发布后通常需要10-30分钟审核 |

### 2.2 内容规则（红线）

**会被限流的行为：**
- 笔记中含有"买""下单""购买"等强营销词（小红书极度反感硬广）
- 图片上有大面积的商品价格/促销信息水印
- 标题党/与内容不符的标题
- 搬运/重复内容
- 笔记质量低（图片模糊、文案敷衍）
- 频繁修改已发布笔记

**小红书的特殊敏感点：**
- 非常反感"微商风"内容
- 要求内容有"真实体验感"——即使用AI生成也要让内容看起来像真人分享
- 健康食品品类会被更严格审核功效宣称
- 评论区不能引导跳转外部平台

**AI内容注意事项：**
- 小红书目前对AI图片/文案的检测较弱
- 但如果多个账号发布相似度过高的内容会被判批量营销
- 建议：每篇图文至少有1张实拍图（哪怕只是手持产品的随手拍）
- 文案风格要"口语化"，不能像AI一样太完美工整

### 2.3 小红书算法偏好

| 指标 | 权重 | 说明 |
|------|------|------|
| 封面图点击率 | 极高 | 封面决定了80%的流量 |
| 完读率 | 高 | 用户是否读完全文 |
| 互动率 | 高 | 收藏>点赞>评论 的权重顺序 |
| 搜索匹配度 | 高 | 标题/正文中的关键词与用户搜索的匹配 |

**小红书的核心逻辑是"搜索+推荐"双引擎：**
- 发布后24小时内主要靠推荐曝光
- 24小时后如果笔记在搜索中排名好，会持续获得长尾流量
- 所以标题中的SEO关键词极其重要

### 2.4 小红书话术转化技巧

**不要说：** "买它""赶紧下单""链接在这"
**要说：** "感兴趣的姐妹可以看看""我把链接放在这里了""自用推荐不踩雷"

小红书的转化话术要伪装成"好友推荐"而不是"销售员叫卖"。

## 三、视频号运营规则

### 3.1 账号基础要求

| 项目 | 要求 |
|------|------|
| 开通视频号小店 | 需要营业执照或个人满足条件 |
| 挂车带货 | 开通小店后可在视频中挂商品 |
| 每日发布上限 | 无明确限制，但建议不超过8条/天 |
| 核心优势 | 微信生态（朋友圈转发、社群裂变、公众号引流） |

### 3.2 内容规则

**视频号审核相对宽松：**
- 对带货内容的容忍度高于小红书
- 但对健康食品的功效宣称同样严格
- 内容质量门槛较低，但低质内容不会获得推荐

**视频号的特殊优势：**
- 朋友圈转发是最强推流信号（比平台内互动还重要）
- 微信群分享可以直接获取精准流量
- 可以在视频号主页设置"关注后自动进群"
- 公众号和视频号可以互相导流

### 3.3 视频号算法偏好

| 指标 | 权重 | 说明 |
|------|------|------|
| 社交推荐 | 极高 | 朋友点赞/转发的内容会推给他的朋友 |
| 完播率 | 高 | 同抖音逻辑 |
| 转发率 | 极高 | 转发到朋友圈/微信群是最强信号 |
| 关注率 | 中 | 影响后续推送 |

**视频号推流机制与抖音的最大区别：**
- 抖音：纯算法推荐为主
- 视频号：社交推荐+算法推荐双引擎
- 意味着：一条视频如果被几个人转发到朋友圈，可能比投DOU+还有效

### 3.4 视频号带货的核心策略

- 内容要"值得转发" — 不是为了卖货，而是让用户觉得"这个对我朋友/家人有用"
- 话术方向：家庭关怀、健康科普、送礼推荐
- 在视频末尾加一句"觉得有用转发给家人"比"赶紧下单"更有效
- 利用视频号的"直播预约"功能，为未来可能的直播带货蓄水

## 六、AI Agent编制

### 6.1 AI Agent矩阵

你们的"AI团队"比人类团队更大。以下是需要部署的AI Agent角色：

```
┌─────────────────────────────────────────────────────┐
│              AI Agent 总控（你通过Claude指挥）          │

              ↓
┌─────────────────────────────────────────────────────┐
│  Agent 1: 策略分析师 (Claude)                         │
│  职责：每日数据分析 → 生成次日策略建议                  │
│  输入：每日数据汇总表                                  │
│  输出：次日生产指令 + 投流建议 + 风险预警               │

┌─────────────────────────────────────────────────────┐
```

### 6.2 人类×AI的协作分工矩阵

| 任务环节 | AI做什么 | 人类做什么 |
|---------|---------|----------|
| 策略决策 | 提供数据分析和建议 | 最终拍板决策 |
| 脚本生成 | 批量生成50条/天 | 10分钟快速筛选，剔除5-10条 |
| 图片生成 | 批量生成50-80张/天 | 1分钟/张质量抽查 |
| 视频生成 | 批量生成30-50段/天 | 检查画面瑕疵 |
| 配音 | 自动生成 | 检查配音质量 |
| 视频合成 | 剪映模板批量套用 | 最终合成确认 |
| 合规检查 | 自动扫描违禁词 | 30秒人工复核 |
| 发布 | （目前需人工） | 按排期手动发布 |
| 数据回收 | 自动抓取（部分） | 补充手动数据 |
| 投流操作 | 出单预测评分 | 根据评分操作DOU+ |
| 评论互动 | 提供话术模板 | 实际回复评论 |
| 每日复盘 | 自动生成分析报告 | 讨论并确认次日方向 |
