## 二、场景×产品×人设 三维矩阵

生产变体的核心逻辑：**同一个产品 × 不同场景 × 不同人设 = 完全不同的内容**

### 2.1 场景库（共10个核心场景）

| 场景ID | 场景名 | 画面描述 | 适用产品 | 适用人设 |
|--------|--------|---------|---------|---------|
| S01 | 办公室下午茶 | 办公桌+电脑+产品 | 罐装/梨膏糖 | 家豪/小瓶子 |
| S02 | 家庭客厅 | 沙发/茶几+产品+家人 | 罐装/礼盒 | 灏灏/小瓶子 |
| S03 | 送礼场景 | 礼盒+包装纸+丝带 | 礼盒¥158/168 | 泽鑫/和煦好物 |
| S04 | 出差/差旅 | 行李箱/酒店桌+产品 | 罐装/梨膏糖 | 家豪/泽鑫 |
| S05 | 健身/运动后 | 运动场景+补给 | 罐装 | 家豪 |

| S10 | 节日/纪念日 | 节日装饰+礼盒 | 礼盒¥158/168 | 泽鑫 |

### 2.2 变体生成公式

每条内容的变体 = 更换以下任意1-3个要素：

```
变体维度1：换hook（同样的内容，换一种开头方式）
变体维度2：换场景（同样的卖点，放在不同使用场景中）
变体维度3：换配音（同样的脚本，换声音风格/语速）
变体维度4：换视觉（同样的脚本，换封面图/产品展示角度）
变体维度5：换CTA（同样的前段，换不同的促单话术）
变体维度6：换产品（同样的模板结构，换成另一款产品）
```

## 四、AI素材+实拍素材混剪指导

### 4.1 混剪比例建议

| 视频类型 | AI素材占比 | 实拍素材占比 | 说明 |
|---------|---------|---------|------|
| 品牌背书型 | 70% | 30% | AI做品牌质感画面，实拍做产品实物 |
| 场景痛点型 | 40% | 60% | 实拍场景更真实，AI补充产品特写 |
| 开箱测评型 | 20% | 80% | 开箱必须实拍才有信服力 |
| 科普植入型 | 80% | 20% | 科普画面AI效率高，穿插实拍产品 |
| 促销限时型 | 50% | 50% | AI做视觉冲击，实拍做真实感 |

### 4.2 实拍素材拍摄清单（每周拍一次）

每人每周花30分钟拍以下素材存入共享库：

```
□ 产品手持特写（手拿产品对着镜头展示，3-5个角度）
□ 开箱全过程（从快递到拆开到取出产品，完整录制）
□ 倒饮过程（开罐/开瓶 → 倒入杯中 → 杯子特写）
□ 使用场景（产品放在办公桌上/茶几上，自然随手拍）
□ 试吃/试饮反应（吃梨膏糖/喝饮料的表情反应）
□ 产品细节（logo特写、配料表特写、包装质感特写）
```

### 4.3 混剪剪辑节奏模板

**15秒视频标准结构：**
```
0-3s:  hook画面（实拍:手拿产品/AI:震撼视觉 → 配AI口播）
3-8s:  卖点展示（AI:产品特写动效+实拍:使用场景 → 交替）
8-12s: 信任建立（实拍:配料表/品牌logo特写 → AI口播）
12-15s: CTA（AI:产品+价格文字 → 配AI口播促单）
```

**25秒视频标准结构：**
```
0-3s:  hook（实拍/AI → 口播）
3-8s:  问题/场景（实拍场景 → 引出产品需求）
8-15s: 产品展示（AI动效+实拍穿插 → 卖点口播）
15-20s: 信任背书（实拍:品牌logo/配料表 + AI:品牌文化画面）
20-25s: CTA促单（AI:产品+价格画面 → 促单口播）
```
